# 🎭 Code Poetry Generator

> *Where algorithms meet artistry*

Transform your code into beautiful poetry. This experimental AI-powered tool analyzes code structure, patterns, and semantics to generate creative poetic interpretations.

## ✨ Features

- **Multi-language support**: Python, JavaScript, TypeScript, Go, Rust, and more
- **Multiple poetry styles**: Haiku, Free Verse, Sonnet, and Abstract
- **Semantic analysis**: Understands code intent and creates meaningful metaphors
- **Beautiful output**: Formatted poetry with syntax highlighting

## 🚀 Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Generate poetry from a code file
python poetry_gen.py examples/fibonacci.py --style haiku

# Or use the CLI
./code-poem examples/algorithm.ts --style free-verse
```

## 📖 Examples

### Input (Python):
```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```

### Output (Haiku):
```
Numbers branch and grow
Like trees reaching for the sky
Golden spirals form
```

## 🎨 Poetry Styles

1. **Haiku** - 5-7-5 syllable structure, captures essence
2. **Free Verse** - Unstructured, flowing interpretation
3. **Sonnet** - 14 lines with rhythmic patterns
4. **Abstract** - Experimental, avant-garde expression

## 🤖 How It Works

1. **Lexical Analysis**: Tokenizes code into meaningful units
2. **Semantic Mapping**: Maps programming concepts to poetic metaphors
3. **Structure Analysis**: Identifies patterns (loops, recursion, conditions)
4. **Creative Generation**: Synthesizes poetry using AI-inspired techniques
