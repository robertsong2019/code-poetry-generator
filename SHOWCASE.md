# 🎭 Code Poetry Generator - Creative Night Showcase

## What I Built Tonight

An experimental AI-powered tool that transforms code into beautiful poetry, exploring the intersection of technology and art.

### 🌟 Live Repository
**https://github.com/robertsong2019/code-poetry-generator**

## Features Implemented

### 1. Multi-Language Code Analyzer
- Uses Pygments for lexical analysis
- Supports Python, JavaScript, TypeScript, Go, Rust, and 500+ languages
- Extracts: functions, loops, conditions, variables, complexity metrics

### 2. Four Poetry Styles

#### 🎋 Haiku (5-7-5)
Perfect for capturing the essence of algorithms:
```
Dance unfolds
In cycles of purposeful steps
Code becomes rhythm
```

#### 🌊 Free Verse
Flowing interpretation of code structure:
```
In the realm of py,
Where quicksort dwells,

Round and round,
Like planets in orbit,
The loops spin their tales.
```

#### 📜 Sonnet
14-line classical poetry:
```
Upon the screen where characters take flight,
A dance emerges from the programmer's sight.
Through cycles that spin with grace and might,
The algorithm dances, purely code-born light.
```

#### 🎨 Abstract
Experimental avant-garde:
```
crossroads . choice . void
ritual . dance . void

>>> [SYNTAX DISSOLVES]
>>> [MEANING EMERGES]
>>> [CODE = POETRY]
```

### 3. Rich CLI Experience
```bash
python poetry_gen.py examples/fibonacci.py --style haiku --show-code
```
- Beautiful terminal output with syntax highlighting
- Progress indicators and styled panels
- Optional file export

## Try It Yourself

```bash
# Clone the repo
git clone https://github.com/robertsong2019/code-poetry-generator
cd code-poetry-generator

# Install dependencies
pip install -r requirements.txt

# Generate poetry
python poetry_gen.py examples/fibonacci.py --style haiku --show-code

# Try all styles
python poetry_gen.py examples/neural_network.py --style sonnet
python poetry_gen.py examples/quicksort.py --style free-verse
python poetry_gen.py examples/binary_search.ts --style abstract
```
